# Linux Multi-User DevOps Server Setup 🚀

## 📌 Project Overview
This project simulates a real production Linux server with multiple users, automation, monitoring, and backup system.

## 🏗️ Features
- Multi-user system (dev, qa, prod)
- File permission management
- Automated monitoring using shell scripts
- Backup system using cron jobs
- Log generation

## 🛠️ Technologies Used
- Linux (Ubuntu)
- Bash scripting
- Cron jobs

## ⚙️ Setup Steps
1. Create users
2. Create directories (/company/dev, qa, prod)
3. Set permissions
4. Add scripts
5. Setup cron jobs
